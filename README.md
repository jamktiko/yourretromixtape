# Your Retro Mixtape

The app draws a random, curated music video based on the genre selected by the user. In addition to the video, the user receives background information or other interesting facts about it. The goal is to offer an additional way to enjoy music videos, allowing users to discover unfamiliar videos while learning something new about songs they already know.

## Description

In Your Retro Mixtape, users can choose from one of the available music genres to receive a randomly selected music video with accompanying fun facts or background information. Alternatively the user can also choose to forego the genre selection, instead receiving a random music video from all the available genres. After receiving a music video, the user can return back to the genre selcetion or keep receiving videos from the same category.

## Getting Started

### Dependencies

This project is build using the following tecnologies:

- SvelteKit: Primary framework for routing.
- Svelte 5: Used for building the user interface.
- Typescript: Provides static typing to ensure code quality and reduce runtime errors.
- ESLint & Prettier: Tools for maintaining consistent code style and preventing common syntax issues.
- Tailwind CSS: CSS framework used for rapid and consistent UI styling.

### Installing

1. Clone the repository:

```
git clone https://github.com/jamktiko/yourretromixtape.git
```

2. Go to the project directory:

```
cd yourretromixtape
```

3. Install dependencies:

```
npm install
```

### Executing program

1. Start the dev server:

```
npm run dev
```

2. Navigate to the website indicated after running the previous command.

## Help

- npm install fails: Ensure that you have Node.js (v18+) installed. You can check your version by running node -v in the terminal.

- Port already in use: If localhost:5173 is busy, you can run the app on a different port using npm run dev -- --port 3000.
