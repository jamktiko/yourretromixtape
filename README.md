# Your Retro Mixtape

The app draws a random, curated music video based on the genre selected by the user. In addition to the video, the user receives background information or other interesting facts about it. The goal is to offer an additional way to enjoy music videos, allowing users to discover unfamiliar videos while learning something new about songs they already know.

### Dependencies

This project is build using the following tecnologies:

- SvelteKit: Primary framework for routing.
- Svelte 5: Used for building the user interface.
- Typescript: Provides static typing to ensure code quality and reduce runtime errors.
- ESLint & Prettier: Tools for maintaining consistent code style and preventing common syntax issues.
- Tailwind CSS: CSS framework used for rapid and consistent UI styling.
